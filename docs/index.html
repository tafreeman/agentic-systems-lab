Agentic Systems Lab

Research and prototyping companion to agentic-runtime-platform. Runnable workflow examples, security-hardening sprints, evaluation patterns, and orchestration ideas — prototyped here before upstreaming to the production runtime.

bash — agentic-systems-lab
$ git clone https://github.com/tafreeman/agentic-systems-lab.git
$ cd agentic-systems-lab && just setup
# no API keys? run key-free in no-LLM mode:
$ AGENTIC_NO_LLM=1 python examples/01_hello_workflow.py
$ agentic list workflows
$ printf '{"input_text":"hello"}\n' > test-input.json
$ agentic run test_deterministic --input test-input.json

Highlights

What's in the lab

🧪

Six runnable examples

Self-contained pipeline, RAG, custom agent, model routing, evaluation, and adapter-switching workflows — all executable key-free with AGENTIC_NO_LLM=1.

⚡

DAG executor

Kahn's algorithm + asyncio FIRST_COMPLETED scheduling with cascade-failure handling, deadlock detection, and SSE/WebSocket streaming.

📡

Tiered model router

Tier 0–5 routing with exponential backoff cooldowns and per-provider circuit breakers — keeps workflows running when a model is unavailable.

📋

YAML workflow DSL

when: / loop_until: control-flow constructs compiled to native DAG or LangGraph — switch execution engines without rewriting workflow definitions.

📊

Rubric evaluation

DORA-inspired S/A/B/C/D/F rubric grading across 8 evaluator types with batch and streaming runners — measure quality before upstreaming.

🗂

6 production workflow definitions

Real YAML workflow definitions used as the canonical integration test surface — edit and run them to explore the DSL before promoting to the runtime platform.

Get started

Install & run

Quick (justfile) 
$ git clone https://github.com/tafreeman/agentic-systems-lab.git
$ cd agentic-systems-lab
$ just setup
Manual (pip) 
$ pip install -e '.[dev,server,langchain]'
Run an example 
AGENTIC_NO_LLM=1 python examples/01_hello_workflow.py
python examples/02_rag_pipeline.py
python examples/04_model_routing.py
CLI workflows 
$ agentic list workflows
$ printf '{"input_text":"hello"}\n' > test-input.json
$ agentic run test_deterministic \
    --input test-input.json
